## 1. 项目架构与完整预期功能清单

### 1.1 项目定位

**运输方案比价与优化智能体** — 物流运费核算与方案推荐系统。用户以自然语言或表单输入运输需求，系统从承运商费率表匹配方案、按 `Cost = max(Min_Cost, Rate × Weight)` 计费、多维度加权评分推荐最优方案。无直达时自动搜索转运路径。

### 1.2 架构分层

```
前端 (Vue 3 + Element Plus)
  App.vue, NLInput.vue, CompareForm.vue, ResultTable.vue,
  RecommendCard.vue, AgentSettings.vue, StatisticsCard.vue,
  ChatPanel.vue, FlowVisualization.vue, HistoryPanel.vue,
  ExportCard.vue, AppHeader.vue
  utils/portUtils.js (港口名称映射工具)

后端 (FastAPI)
  main.py (18个API端点, 启动初始化, 报告生成)
  config.py, freight_service.py, graph_router.py,
  llm_service.py, models.py, tools.py,
  database.py, db_models.py, init_db.py,
  base_data_store.py, common/data_cleaner.py

数据层
  CSV 文件 (FreightRates.csv/combined/extended)
  SQLite (freight.db, 由 init_db.py 生成)
```
